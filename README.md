
# Moveo project

Online web application to share pieces of code between Mentor and Student.

## Run Locally

Go to the backend directory 

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run devStart
```

Go to the frontend directory 

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
